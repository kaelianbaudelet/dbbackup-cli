DBBackup CLI

Un outil Bash puissant et moderne pour automatiser, chiffrer et transférer vos sauvegardes MariaDB/MySQL.

Copié dans le presse-papiers !

Installation rapide

$ curl -o install.sh https://raw.githubusercontent.com/kaelianbaudelet/dbbackup-cli/main/install.sh && chmod +x install.sh && sudo ./install.sh

Exécutez cette commande sur n'importe quel système Linux pour installer le CLI globalement.

⚡

Automatisé

Planifiez vos sauvegardes en quelques secondes avec le module schedule intégré.

🔒

Chiffré

Sécurisez vos données avec le chiffrement AES-256 via OpenSSL.

☁️

Distant

Transférez automatiquement vos backups vers un serveur distant via SCP.

🛠️

Complet

Gestion des logs, rotation des fichiers et restauration en une commande.

Commandes courantes

Faire une sauvegarde immédiate
dbbackup backup -u root -p secret -d matable
Planifier une sauvegarde (tous les jours à 2h)
dbbackup schedule add -n "daily" -c "0 2 * * *" -u root -p secret -d mydb
Lister les sauvegardes effectuées
dbbackup backup list
Restaurer une base de données
dbbackup restore -f backup_2024.sql.gz -d mydb

DBBackup CLI - Créé avec ❤️ pour les administrateurs système

Désinstaller : sudo /usr/local/lib/dbbackup-cli/uninstall.sh
