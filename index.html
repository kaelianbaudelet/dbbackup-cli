Automate backups.

A simple, reliable Bash CLI to backup, encrypt, and transfer MariaDB/MySQL databases. Built for stability and speed.

Quick Installation
curl -o install.sh https://raw.githubusercontent.com/kaelianbaudelet/dbbackup-cli/main/install.sh && chmod +x install.sh && sudo ./install.sh

Capabilities

  • Native Scheduler

    Built-in cron management. Schedule backups with standard cron expressions without leaving the CLI.

    dbbackup schedule add -n "daily" -c "0 2 * * *"
  • AES-256 Encryption

    Encrypt sensitive dumps locally using OpenSSL before any transfer. Secure by default.

    dbbackup backup --encrypt -d production_db
  • Remote Transfer

    Automated SCP/SSH transfers to remote storage nodes with key-based authentication support.

Quick Start

  • Create a backup

    Dumps the database, compresses it with gzip, and stores it in /usr/local/lib/dbbackup-cli/backups.

    dbbackup backup -u root -p secret -d mydb
  • Restore data

    Decompress and import a specific backup file into a target database.

    dbbackup restore -f backup_file.sql.gz -d mydb
  • Check logs

    View the operation history and verify that automated tasks ran successfully.

    dbbackup log list
